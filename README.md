# SmartTrack - Academic Credit & Course Planning System

A fully deployed academic tracking platform that helps students manage and visualise their academic progress.

## Live Demo
https://smarttrack-omega.vercel.app

## Features
- Track completed and remaining credits
- Basket-wise course analysis
- Semester planner with credit load calculator
- Course history with GPA calculation
- Academic calendar with reminders
- CSV export of course data
- Charts and visualizations

## Tech Stack
- Next.js 16
- Tailwind CSS
- Supabase (Database + Auth)
- Recharts
- Vercel (Deployment)

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Create `.env.local` with your Supabase credentials
4. Run `npm run dev`
